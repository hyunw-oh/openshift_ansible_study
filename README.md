openshift_ansible_study
=======================
오픈 시프트 기초에 대해서 공부한 것을 기록으로 남기려고 한다.
오픈시프트는 레드햇의 PaaS 서비스로 엔터프라이즈 버전과 넷 버전 그리고 프라이빗 서비스를 위 오픈소스버전(일명 오리진)으로 관리하고 있는 클라우드 인프라 툴이다.
레드헷에서는 오픈시프트를 엔터프라이즈 kubernetes라고 이야기한다.
사실 코어부분은 쿠버네티스엔진이 대부분 컨트롤 해주고 있다. 
다만 차이점이 있다면 젠킨스처럼 변경한 소스를 git에 올리기만 해도
자동으로 컨테이너에 적용되서 다시 빌드해서 서비스한다.

이럴 때 간결하게 어플리케이션의 라이프사이클을 관리해주는 것도 오픈시프트의 특징이다.
새로운 빌드버전이 완전히 올라오기 전에는 이전 버전의 어플리케이션을 죽이지 않고
유지하고 있다가 새로운 버전이 완전히 올라오면 그제서야 이전 버전의 앱을 교체하는 특징을 갖는다.
